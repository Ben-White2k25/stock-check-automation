# Stock Check Automation & Dashboard Project

## Overview
This project simulates a warehouse stock-checking system using Python, Power BI, and GitHub.  
It identifies:
- Quantity mismatches  
- Duplicate Item IDs  
- Missing labels  
- General stock inconsistencies  

It then generates a **CSV error report** that can be imported into **Power BI** for dashboard visualisation.

---

## Tools Used
- Python
- Power BI
- GitHub
- Pandas

---

## How the Script Works
1. Loads a mock dataset (or real warehouse data)
2. Checks for:
   - Quantity mismatches
   - Duplicate Item IDs
   - Missing labels
3. Outputs results to the terminal
4. Creates `stock_error_report.csv` for Power BI
